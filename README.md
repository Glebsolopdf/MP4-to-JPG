Как работает скрипт:

1. При запуске скрипт ищет все .mp4 из директории в которой лежит
2. Обрабатывает центральный кадр в видео и превращает его в .jpg
3. На выходе вы получаете скрин из видео без потери качества

Важно что скрипт обрабатывает абсолютно **ВСЕ** .mp4 которые лежат в одной директории с ним
